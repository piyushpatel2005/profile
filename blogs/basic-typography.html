Basic Typography

HTML typography involves semantic tags for different type forms. There are different types of tags to represent strong emphasis, heading, paragraph, marking and do you remember Maths and certain equations which required subscripts and superscripts??? There is a specific tag for them as well! Let's go through them one by one.

You can use headings from h1 to h6. Remember, these tags are given semantic order. I mean, h1 is the main heading, h2 is the subheading and h3 is sub to subheading and so on. So, the article will be given main heading tag h1. However, there can be more than h1 tags in a single page. For example, the first heading might be page heading then for each article, there can be separate h1s.
